2. Complexidade Ciclomática
   
   A complexidade ciclomática é calculada a partir da fórmula M = E - N + 2P, sendo:

   E = Número de Arestas;
   
   N = Número de Nós;
   
   P = Número de componentes conectados;
   
   M = Complexidade Ciclomática;
   


   No grafo de fluxo acima:
   
   E = 10;
   
   N = 10;
   
   P = 1;
   
   M = 2;
